# Features — Portfolio Minimal

## Core Features

### Book Cover Opening (Phase 2 — IN PROGRESS)
- Full-screen black book cover on load
- "AAMIR ALI" in gold Cormorant Garamond
- "Not a resume. A record." tagline
- Cinematic center-split animation (left panel slides left, right slides right)
- Reveals chapter nav underneath
- Trigger: scroll or click

### Chapter Navigation (Phase 2)
- "In This Issue" table of contents
- Click any chapter to jump there with page-fold transition
- Persistent chapter indicator (corner overlay showing current chapter)

### Page-Fold Transitions (Phase 2)
- Framer Motion 3D perspective fold between chapters
- Feels like physically turning a page
- Triggered on scroll or chapter nav click

### Chapter 01: The Background
- Two-column magazine article layout
- Bio paragraphs styled like a print magazine article (Lora serif)
- Pull quote in gold italic
- Skills tags styled like magazine sidebar

### Chapter 02: The Toolkit
- 8 AI/automation tool cards in editorial grid
- Tool categories as magazine-style labels
- Tool descriptions in Lora serif

### Chapter 03: The Work
- 6 project cards — magazine feature layout
- Project #, status badge, stack tags
- GitHub links

### Chapter 04: The Record (CV)
- Embedded inline — no separate page
- Career timeline (animated horizontal scroll on desktop, vertical on mobile)
- Two-column: experience + skills sidebar
- Print-optimized styling

### Chapter 05: Work With Me
- 9 service cards
- 4-step process section (01–04)
- Contact form (Web3Forms) embedded
- All contact links with Lucide icons

### Global UI
- Custom cursor (dot + trailing ring)
- Scroll progress bar (gold, 2px top)
- Navbar: chapter-aware, shows current chapter name
- Footer: minimal, with chapter links
